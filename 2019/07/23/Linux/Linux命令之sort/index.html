在一些临时性排序需求的时候，使用该命令可以快速的处理。小巧的工具使我们使用非常便捷。
语法：sort [-bcCdfghiRMmnrsuVz] [-k field1[,field2]] [-S memsize] [-T dir] [-t char] [-o output] [file ...]
